# wbms
web site back_end management system

_core: SpringMVC + Shiro + Mybatis-plus_

other: ehcache, freemarker, maven, adminLTE2, druid, logback, tomcat.

-  **项目构成** 

1. 系统后台使用 Spring + SpringMVC
2. 权限控制使用shiro
3. 缓存使用ehcache
4. 前端模板使用 freemarker + adminLTE2
5. 日志logback
6. 数据库连接池 druid
7. 持久层 mybatis-plus
8. 容器 tomcat
